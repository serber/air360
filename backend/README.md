# Air360 API Backend

Fastify-based API backend for Air360 device registration, measurement ingest,
and data storage.

This directory contains the backend source code. Backend documentation lives in
[`../docs/backend/README.md`](../docs/backend/README.md).

## Project

- Runtime: Node.js >= 20
- Language: TypeScript
- Framework: Fastify 5
- Database: PostgreSQL + TimescaleDB

Use the backend docs for setup, commands, migrations, data model, API reference,
deployment notes, and source-of-truth guidance.
