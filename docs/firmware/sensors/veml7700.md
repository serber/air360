# VEML7700

Ambient light sensor from Vishay with an I2C interface.

## Transport

- I2C, bus 0 (SDA=GPIO8, SCL=GPIO9)
- Address: `0x10` (fixed)
- Clock: 100 kHz, pull-ups enabled

## Initialization

1. Initialize the `i2cdev` subsystem
2. Initialize the sensor descriptor via `veml7700_init_desc()`
3. Verify device presence via `veml7700_probe()`
4. Configure the sensor:
   - Gain: `VEML7700_GAIN_DIV_8` (1/8 — extends range toward high illuminance)
   - Integration time: `VEML7700_INTEGRATION_TIME_100MS`
   - Persistence protection: `VEML7700_PERSISTENCE_PROTECTION_1`
   - Interrupts: disabled
   - Shutdown: disabled (sensor active)
   - Thresholds: zeroed (not used)
   - Power saving mode: `VEML7700_POWER_SAVING_MODE_500MS` (configured but disabled)
5. Apply configuration via `veml7700_set_config()`

## Polling

Each poll cycle:
1. Read illuminance via `veml7700_get_ambient_light()` — returns a `uint32_t` lux value
2. Cast to `float` and record

## Measurements

| Measurement | ValueKind | Unit |
|-------------|-----------|------|
| Illuminance | `kIlluminanceLux` | lux |

## Notes

- Gain 1/8 extends the effective range toward direct sunlight. Accuracy at low indoor illuminance is reduced as a result.
- Integration time of 100 ms is a compromise between sensitivity and update rate.
- Power saving mode is configured but disabled — the sensor runs in continuous measurement mode.
- The reading is a single integer lux value with no fractional part.
- If the read call returns an error, `initialized_` is reset.

## Recommended poll interval

Minimum 5 seconds. The actual sensor acquisition takes integration time (100 ms) plus internal processing.
