# Ecosystem Docs

This directory contains ecosystem-facing analysis documents for Air360.

These files are not implementation truth for the firmware. They are planning and positioning documents that compare Air360 with adjacent ecosystems, identify integration opportunities, and help prioritize work that improves compatibility or adoption.

## Document Map

- [`sensor-community-opportunity-roadmap.md`](sensor-community-opportunity-roadmap.md)
  Review of the full open `opendata-stuttgart/sensors-software` issue tracker and a prioritized list of areas where Air360 can deliver compatible or adjacent features with the highest practical value.
- [`sensor-community-issues-prs-review-2026-04-14.md`](sensor-community-issues-prs-review-2026-04-14.md)
  Second-pass review of all open `sensors-software` issues and open PRs, filtered against the current Air360 firmware to produce a concrete implementation task list.
